#Executar Solutions no Microsoft Visual Studio Express 2012 - windows desktop



1 - Open Project

Vá em File -> OpenProject -> caminhoDoDiretorio_Solution(o caminho onde foi baixado as questoes)

obs: escolha os arquivos com extensoes:.csproj.

2- Selecione o arquivo nome_arquivo.cs (normalmente o nome padrao é Program.cs)

Esse arquivo é responsavel por carregar sua aplicacao.

3 - Vá em Project -> nome_projeto.properties ->Debug

No campo Command arguments digite o numero correspondente a solution que gostaria de executar, por exemplo: No
 projeto "Questoes  de 1-10" digite no campo a faixa de valores entre 0 e 9 para obter  a execucao quando for dada a ordem
de iniciar o  projeto.

4 - Clique em Start

Clicando em start, depois de alguns instantes aparecerá a solution correspondente ao numero escolhido.
